## Rust必背

- Rust 函数参数的类型和返回值的类型都必须显式定义，如果没有返回值可以省略，返回 unit。函数内部如果提前返回，需要用 return 关键字，否则最后一个表达式就是其返回值。如果最后一个表达式后添加了; 分号，隐含其返回值为 unit。
- Trait 函数是指第一个参数不是self关键字的任意函数。Trait 函数可以通过 trait 或者实现类型的命名空间来调用。
